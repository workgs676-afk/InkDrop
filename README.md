## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🌐 Usage

Open your browser and go to:

```
http://localhost:3000
```

Start creating and managing your notes!
